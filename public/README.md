# scroll.chat

Install:

```html
<script src="https://unpkg.com/scroll.chat@latest/bundle.js" /></script>
```

```
npm install scroll.chat
```

```
yarn add scroll.chat
```

Embed:

```html
<!DOCTYPE html>
<html>
  <!-- ... -->
  <div style="position: absolute;">
    <!-- your page content -->
  </div>
  <scroll-chat />
</html>
```
