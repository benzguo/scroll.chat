scroll.chat
Add a live chat overlay to ~any page
live – colored squares in the left sidebar represent live viewers.
anonymous – no logins, anyone can chat.
ephemeral – every URL gets its own chat room, and can store up to 100 messages.
customizable – sort of. there's a dark theme, and the chat can be closed by default.
open source – contributions welcome.
decentralized – powered by GunDB.
