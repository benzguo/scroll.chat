# P2P web component

GunDB realtime p2p chat in a Svelte web component

## yarn commands

- `build`: build to bundle.js
- `dev`: build/watch/serve
- `server`: run gundb server

## clear gunDB user

browser console:
`sessionStorage.clear()`
